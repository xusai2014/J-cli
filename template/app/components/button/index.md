# Button

## 实例


```jsx
import { Button } from 'sxp-rnc';

<Button onPress={() => {}}>点击</Button>
```

## 属性说明

| **属性** | **说明** | **类型** | **可选值** | **默认值** |
| --- | --- | --- | --- | --- |
| onPress | 点击事件函数 | Function | - | - |
| disabled | 是否禁用按钮 | Boolean | true \| false | false |
| style | 按钮容器样式 | Object | - | - |
| textStyle | 按钮文字样式 | Object | - | - |
| themeColor | 按钮主题颜色 | String | - | #108ee9 |
| ghost | 是否显示透明背景 | Boolean | - | false |
